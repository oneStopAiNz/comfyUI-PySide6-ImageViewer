# ComfyUI Image Explorer

Build a PySide6 desktop application to browse and analyze images generated by ComfyUI.

## Core Features

### Image Gallery
- Load images from folder
- Thumbnail grid
- Thumbnail disk caching
- Lazy loading

### Image Viewer
- Display full resolution image
- Mouse zoom and pan
- Preserve zoom and pan when switching images
- Keyboard navigation (left/right)

### Workflow Inspector
- Extract workflow JSON from PNG metadata
- Sidebar showing nodes
- Node parameter viewer (key/value table)

### Parameter Comparison
- Detect parameter changes between images
- Highlight differences in UI

## Performance Requirements

- Support thousands of images
- Async thumbnail generation
- Non-blocking UI
